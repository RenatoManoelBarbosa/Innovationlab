PACOTE DE HOSPEDAGEM
Sala de Inovacao e Inteligencia Organizacional da PMMG

Arquivos principais:
- index.html: pagina institucional principal.
- sala-inovacao-com-quadro-colaboradores.jpg: imagem usada no hero e no encerramento.
- sala-inovacao-com-quadro-colaboradores.png: imagem-base preservada para compatibilidade e uso no tour.
- tour-3d/index.html: visualizacao panoramica interativa.
- tour-3d/sala-inovacao-com-quadro-colaboradores.png: imagem usada no tour 3D.

Como hospedar:
1. Envie todo o conteudo desta pasta para a raiz da hospedagem.
2. Mantenha a estrutura de pastas exatamente como esta.
3. A pagina inicial sera carregada pelo arquivo index.html.
4. O tour 3D ficara disponivel em /tour-3d/index.html.

Observacao:
O projeto nao depende de bibliotecas externas. O HTML, CSS e JavaScript estao embutidos nos arquivos.
